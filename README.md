## Overview

**CtrlA** is a web application designed for managing team attendance. It allows users to easily register, visualize, and export attendance records for events, classes, and work shifts.

---

## Main Features

- Register attendance for users in various events
- Search and view user and event lists
- Export attendance reports
- Intuitive and modern web interface

---

## Technologies Used

- **Java** — backend logic and API
- **JavaScript** — frontend interactivity
- **HTML & CSS** — interface layout and design
- **Dockerfile & Batchfile** — automated deployment and setup
